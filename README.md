# Diplomado Superior en Redes Neuronales Artificiales y Deep Learning

Repositorio que contiene materiales, código y ejercicios desarrollados
durante el **Diplomado Superior en Redes Neuronales Artificiales y Deep
Learning**.

## Descripción

Las **Redes Neuronales Artificiales (RNA)** son modelos computacionales
inspirados en el funcionamiento del cerebro humano. Estas redes permiten
que los sistemas aprendan patrones complejos a partir de datos.

El **Deep Learning** es una rama del aprendizaje automático que utiliza
redes neuronales profundas para resolver problemas complejos como
clasificación de imágenes, procesamiento de lenguaje natural,
reconocimiento de voz y predicción de datos.

Este repositorio reúne ejemplos prácticos y ejercicios relacionados con
estos conceptos.

## Objetivos del diplomado

-   Comprender los fundamentos matemáticos de las redes neuronales.
-   Implementar modelos de aprendizaje automático.
-   Diseñar y entrenar redes neuronales profundas.
-   Analizar el rendimiento de modelos de deep learning.
-   Aplicar técnicas de inteligencia artificial en problemas reales.

## Temas principales

### Fundamentos de Machine Learning

-   tipos de aprendizaje
-   modelos supervisados y no supervisados
-   preparación de datos

### Redes Neuronales Artificiales

-   neurona artificial
-   perceptrón
-   funciones de activación
-   redes multicapa

### Entrenamiento de redes

-   función de pérdida
-   optimización
-   backpropagation
-   descenso de gradiente

### Deep Learning

-   redes neuronales profundas
-   redes convolucionales (CNN)
-   redes recurrentes (RNN)
-   regularización y optimización

### Evaluación de modelos

-   métricas de desempeño
-   validación de modelos
-   ajuste de hiperparámetros
